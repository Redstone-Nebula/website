JMCL

红石星云出品的轻量级 Minecraft Java 启动器。快速启动、版本管理、Mod 整合，为每一位玩家打造流畅的 Minecraft 启动体验。

/ 功能特性

启动器特色

为 Minecraft 玩家量身打造的完整功能

极速启动

采用轻量级 Java 启动引擎，启动速度快、内存占用低，让你用更少的时间等待，用更多的时间游戏。

版本管理

支持 Vanilla、Forge、Fabric、NeoForge 等主流加载器，一键下载与版本切换，管理多个游戏版本互不干扰。

Mod 与整合包

支持 Mod 快速导入与整合包管理，无论是 CurseForge 还是 Modrinth，轻松打造属于自己的模组世界。

账户支持

支持 Microsoft 账户登录与离线模式，多种验证方式自由切换，满足正版玩家与局域网联机玩家的不同需求。

跨平台

基于 Java 构建，兼容 Windows、macOS、Linux 三大主流操作系统，一次编写，处处运行。

开源自由

完全开源、免费使用，基于社区共建，你也可以参与代码贡献或二次开发，打造属于你的定制版本。
