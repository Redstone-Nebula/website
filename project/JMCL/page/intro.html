关于 JMCL

深入了解 JMCL 的设计理念、技术架构与完整功能

/ 项目背景

为什么要做 JMCL？

在 Minecraft 社区中，启动器是每位玩家的第一道入口。我们希望为玩家提供一款更轻、更快、更自由的启动器 —— 它既不需要臃肿的安装包，也不需要学习复杂的配置，同时对多平台与多版本有着良好的兼容性。

JMCL（Java Minecraft Launcher）基于 Java 语言构建，使用现代 UI 框架打造，兼具启动速度与稳定性。它既能作为普通玩家的日常启动器，也可以作为开发者学习和二次定制的基础项目。

/ 核心功能

功能总览

一键启动

选择你的游戏版本，点击开始即可进入 Minecraft 世界。整个过程只需几秒钟，告别漫长等待。

多版本管理

支持同时管理多个游戏版本与模组加载器，在 Vanilla、Forge、Fabric、NeoForge 之间无缝切换。

模组与资源包

支持 Mod 拖拽安装、资源包快速启用，整合包一键导入，轻松构建专属的游戏环境。

灵活账户

支持 Microsoft 账户登录，也支持自定义离线角色和第三方服务器验证，多种方案自由选择。

跨平台支持

Windows、macOS、Linux 三端通用，统一的操作体验，无论你在哪种设备上都能顺畅使用。

开源自由

采用开源协议发布，任何人都可以自由使用、修改和分发，社区贡献与反馈都是项目的一部分。
